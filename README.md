# DrawIT – AI Assisted Drawing Web App

DrawIT is an AI-assisted drawing web application built using React.js and Python.

## Features
- Interactive drawing interface
- AI-assisted recognition
- Responsive UI
- Fast deployment

## Technologies Used
- React.js
- Python
- HTML
- CSS
- JavaScript

## Deployment
- Frontend deployed on Vercel
- Backend deployed on Render
